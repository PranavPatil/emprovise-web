• Secure Socket Layer: An Overview

    Secure Socket Layer (SSL) is used for secure communication over the internet. It includes other protocols such as TLS Record Protocol, TLS Handshake Protocol and Alert Protocol. Also some prime TLS attacks are discussed.

  • Security Attacks

    Security attacks are prevalent in the growing internet world. The prime attacks on the application include incomplete Data validation, revealing informative messages, URL redirection, cross-site scripting, invalid session and cookie management, faulty debugging etc.
